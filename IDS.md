# ID элементов страницы

| ID | Описание |
|---|---|
| `main-grid` | Мастер-сетка layout (контейнер grid) |
| `badge-input-card` | Карточка ввода значков |
| `badge-input-wrapper` | Обёртка поля ввода значков |
| `total-badges` | Поле ввода количества значков |
| `badge-controls` | Блок контролов (шт./спиннеры) |
| `badge-spinner` | Контейнер кнопок-стрелок |
| `badge-up-btn` | Кнопка ▲ увеличить |
| `badge-down-btn` | Кнопка ▼ уменьшить |
| `tax-card` | Карточка налога рынка |
| `tax-toggle-wrapper` | Обёртка переключателя налога |
| `apply-tax` | Чекбокс налога 10% |
| `price-controls-card` | Карточка управления ценами |
| `price-buttons-row` | Ряд кнопок Сбросить / Обнулить |
| `reset-prices-btn` | Кнопка Сбросить |
| `clear-prices-btn` | Кнопка Обнулить |
| `sync-btn` | Кнопка Обновить цены (Live) |
| `sync-spinner` | Спиннер синхронизации |
| `sync-btn-text` | Текст кнопки синхронизации |
| `game-week-indicator` | Индикатор текущей недели |
| `base-prices-card` | Карточка базовых цен рынка |
| `base-prices-container` | Контейнер списка базовых цен |
| `right-simulation-column` | Правая колонка (симулятор) |
| `smart-holding-panel` | Блок рекомендаций удержания |
| `result-card` | Блок результата по периоду |
| `result-header` | Заголовок результата (с селектором) |
| `simulation-period` | Селектор периода |
| `best-deal-box` | Контейнер лучшей сделки |
| `offers-section` | Секция вкладок и сетки предложений |
| `week-tabs-wrapper` | Обёртка табов недель |
| `week-tabs` | Навигация табов |
| `tab-all`, `tab-1`..`tab-4`, `tab-0` | Кнопки табов |
| `offers-grid` | Сетка карточек предложений |
| `sync-status` | Статус-бар синхронизации (header) |
| `sync-indicator` | Индикатор статуса (точка/текст) |
| `status-ping` | Анимация пинга |
| `status-dot` | Точка статуса |
| `status-text` | Текст статуса |
| `last-sync-elapsed` | Время с последней синхронизации |
| `next-sync-countdown` | Таймер до следующей синхронизации |
| `toast` | Всплывающее уведомление |
| `toast-message` | Текст уведомления |
| `detail-modal-overlay` | Затемнение/оверлей модального окна детального анализа |
| `detail-modal` | Контейнер модального окна |
| `detail-modal-content` | Внутренний контент модалки (заполняется через JS) |
| `best-deal-image-block` | Блок изображения ресурса в лучшей сделке |
| `settings-container` | Контейнер кнопки и панели настроек |
| `settings-btn` | Кнопка открытия настроек |
| `settings-panel` | Панель настроек (плотность / масштаб) |
| `density-options` | Группа кнопок выбора плотности |
| `scale-options` | Группа кнопок выбора масштаба |
| `view-toggle` | Переключатель режима отображения сетки предложений |
| `view-cards` | Кнопка режима «Карточки» |
| `view-tiles` | Кнопка режима «Плитки» |
| `view-list` | Кнопка режима «Список» |
| `footer` | Футер страницы (логотип, ссылки, дисклеймер) |
