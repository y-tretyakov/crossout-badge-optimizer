# Crossout Badge Optimizer

## 🚀 Aperçu

**Crossout Badge Optimizer** est une application web légère qui aide les joueurs à maximiser la valeur de leurs badges de mécanicien.

L'outil compare automatiquement toutes les options d'échange disponibles, applique la taxe du marché, calcule la rentabilité et détermine quelle option offre le meilleur rendement.

## Fonctionnalités

* Calcul instantané de rentabilité
* Prise en charge de la taxe du marché
* Classement des échanges
* Analyse des gains par badge
* Ajout d'éléments personnalisés
* Prix du marché modifiables
* Interface responsive
* Fonctionnement 100 % côté client

## 📊 Comment ça marche

Pour chaque option d'échange disponible, l'application calcule :

```text
Valeur nette du marché ÷ Coût en badges
```

L'optimiseur prend en compte :

* Quantité de ressource
* Coût en badges
* Prix du marché actuel
* Déduction de la taxe du marché
* Profit net
* Ratio Coins/par-badge
* Score d'efficacité global

Les résultats sont triés automatiquement de l'option la plus rentable à la moins rentable.

## 📦 Installation

Clonez le dépôt :

```bash
git clone https://github.com/YOUR_USERNAME/crossout-badge-optimizer.git
```

Allez dans le dossier du projet :

```bash
cd crossout-badge-optimizer
```

Ouvrez :

```text
index.html
```

dans votre navigateur.

## 🎮 Utilisation

1. Ouvrez l'application.
2. Entrez les prix du marché actuels.
3. Spécifiez les badges de mécanicien disponibles.
4. Consultez le classement de rentabilité.
5. Échangez les badges en utilisant l'option la plus rentable.

## 📈 Exemple

| Ressource    | Coût en badges | Valeur marché | Coins/Badge |
| ------------ | -------------: | ------------: | ----------: |
| Plastic      |            300 |           540 |        1.80 |
| Electronics  |            800 |          1480 |        1.85 |
| Batteries    |            600 |           980 |        1.63 |

L'optimiseur identifie automatiquement l'échange le plus rentable.
