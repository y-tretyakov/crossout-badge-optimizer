# Crossout Badge Optimizer

## 🚀 Überblick

**Crossout Badge Optimizer** ist eine leichte Webanwendung, die Spielern hilft, den Wert ihrer Mechaniker-Abzeichen zu maximieren.

Das Tool vergleicht automatisch alle verfügbaren Tauschoptionen, berücksichtigt Marktsteuern, berechnet die Profitabilität und ermittelt, welche Option die höchste Rendite liefert.

## Funktionen

* Sofortige Rentabilitätsberechnung
* Unterstützung der Marktsteuer
* Vergleich aller Tauschoptionen
* Analyse „Coins pro Abzeichen“
* Benutzerdefinierte Einträge
* Bearbeitbare Marktpreise
* Responsives Design
* Läuft vollständig im Browser

## ✨ Funktionen

* Echtzeit-Berechnungen der Profitabilität
* Coins-pro-Abzeichen Effizienzanalyse
* Automatische Rangfolge der Tauschoptionen
* Unterstützung der Marktsteuer (10%)
* Bearbeitbare Marktpreise
* Unterstützung für benutzerdefinierte Ressourcen
* Responsives Nutzerinterface
* Schnell und ressourcenschonend

## Technologien

* HTML5
* Tailwind CSS
* Vanilla JavaScript

## 📊 Funktionsweise

Für jede verfügbare Tauschoption berechnet die Anwendung:

```text
Netto Marktwert ÷ Abzeichenkosten
```

Der Optimierer bewertet:

* Ressourcenmenge
* Abzeichenkosten
* Aktuellen Marktwert
* Abzugssteuer des Marktes
* Nettogewinn
* Coins-pro-Abzeichen Verhältnis
* Gesamteffizienz

Ergebnisse werden automatisch von der profitabelsten bis zur am wenigsten profitablen Option sortiert.

## 📦 Installation

Klonen Sie das Repository:

```bash
git clone https://github.com/YOUR_USERNAME/crossout-badge-optimizer.git
```

Wechseln Sie in das Projektverzeichnis:

```bash
cd crossout-badge-optimizer
```

Öffnen Sie:

```text
index.html
```

im Browser.

## 🎮 Nutzung

1. Öffnen Sie die Anwendung.
2. Geben Sie aktuelle Marktpreise ein.
3. Geben Sie verfügbare Mechaniker-Abzeichen an.
4. Überprüfen Sie das Profitabilitäts-Ranking.
5. Tauschen Sie Abzeichen anhand der besten Option.

## 📈 Beispiel

| Ressource    | Abzeichen-Kosten | Marktwert | Coins/Abzeichen |
| ------------ | ---------------: | ---------: | ---------------: |
| Plastic      |              300 |       540 |            1.80 |
| Electronics  |              800 |      1480 |            1.85 |
| Batteries    |              600 |       980 |            1.63 |

Der Optimierer identifiziert automatisch die profitabelste Tauschoption.
